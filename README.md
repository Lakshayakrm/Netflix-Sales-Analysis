Netflix Sales Analysis Dashboard

Overview

The Netflix Sales Analysis Dashboard is an interactive Power BI project designed to analyze Netflix content across different countries, genres, release years, durations, and IMDb ratings. The dashboard presents key performance indicators and visual insights that help users understand content distribution, popularity, and rating trends through an intuitive and interactive interface.

Objectives
- Analyze Netflix content by country and genre.
- Compare IMDb ratings across different content durations.
- Track content distribution by release year.
- Identify the highest-rated titles.
- Enable interactive exploration using filters and slicers.

Dashboard Features

KPI Cards displaying:
- Total Genres
- Average IMDb Rating
- Total Directors

Visualization 
- Country-wise Title Distribution
- Genre-wise Content Analysis
- IMDb Rating by Duration
- Maximum IMDb Rating Gauge
- Country → Genre → Type Decomposition Tree
- IMDb Rating by Release Year Scatter Plot
- Interactive Genre Slicer

Tools & Technologies
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling

Key Insights
- Compare Netflix content availability across multiple countries.
- Analyze the popularity of different genres.
- Understand IMDb rating trends based on content duration.
- Explore release year patterns.
- Drill down into country, genre, and content type for detailed analysis.
